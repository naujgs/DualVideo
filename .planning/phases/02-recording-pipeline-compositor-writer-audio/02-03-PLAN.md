4. Update `stopRecording` to use `UIApplication.beginBackgroundTask` for safe finalization (D-06 background task):

```swift
@MainActor
func stopRecording(completion: @escaping (URL?) -> Void = { _ in }) {
    guard case .recording = phase else {
        logger.warning("RecordingManager.stopRecording() called outside .recording phase")
        completion(nil)
        return
    }

    timerTask?.cancel()
    timerTask = nil
    phase = .finalizing

    // Acquire background task to ensure finalization completes even if app backgrounds
    let bgTask = UIApplication.shared.beginBackgroundTask(withName: "finalize-recording") {
        // OS-triggered expiration: cancel to avoid corruption
        logger.error("RecordingManager: background task expired — cancelling recorder")
        self.recorder.cancelAndDiscard()
        UIApplication.shared.endBackgroundTask(.invalid)
    }

    recorder.stopAndFinalize { [weak self] url in
        DispatchQueue.main.async { [weak self] in
            guard let self else { return }
            self.pendingFileURL = url
            self.phase = .idle
            self.elapsedSeconds = 0
            UIApplication.shared.endBackgroundTask(bgTask)
            logger.info("RecordingManager: finalized, bgTask ended, url=\(url?.lastPathComponent ?? "nil")")
            completion(url)
        }
    }
}
```

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| dataOutputQueue → AVAssetWriter audio appends | CMSampleBuffer from mic hardware appended to audio track |
| background task → file system | finishWriting() completes and leaves .mov in temporaryDirectory |
| OS interruption → RecordingManager | External OS events (phone call, background) trigger finalization |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-03-01 | Information Disclosure | Orphaned .mov in `temporaryDirectory` after crash | mitigate | `RecordingManager.init()` calls `cleanUpOrphanedTempFiles()` — scans `temporaryDirectory` for `.mov` files and deletes them before a new session. |
| T-02-03-02 | Denial of Service | Background task expiration before finalization completes | mitigate | `beginBackgroundTask` expiration handler calls `recorder.cancelAndDiscard()` to ensure no partial-corrupt file is left and no resource leak. `endBackgroundTask` always called (in both success and expiry paths). |
| T-02-03-03 | Tampering / Crash | Audio CMSampleBuffer appended while writer is in `.failed` state | mitigate | `appendAudioBuffer` guards `w.status == .writing && aInput.isReadyForMoreMediaData` before every append. Failed appends are logged, not silently ignored. |
